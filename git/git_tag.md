# git tag

### 查看所有标签
> git tag

### 查看指定标签
> git show [标签名]

### 搜索标签
> git tag -a [标签名正则]

### 打标签 -轻量级
> git tag [标签名]

### 打标签 -含附注
> git tag -a [标签名] -m "说明"

### 打标签 -后补标签
> git tag -a [标签名] -m "说明" [commit id]

### 删除标签
> git tag -d [标签名]

### 推送一个标签
> git push [仓库名] [标签名]

### 推送所有标签
> git push [仓库名] --tags
