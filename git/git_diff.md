## git diff

### 比较 工作区 和 暂存区
> git diff   

### 比较 工作区 和 HEAD
> git diff HEAD

### 比较 暂存区 和 HEAD
> git diff --staged

### 比较 工作区 和 某一commit
> git diff \<commit> 

### 比较 commit1 和 commit2
> git diff \<commit1> \<commit2> 

### 显示摘要 而非详情
> git diff --stat ...
